# 📦 Shop Listing

Aplicación web de e-commerce enfocada en la visualización de productos y gestión básica de un carrito de compras. El proyecto simula una tienda online moderna, permitiendo explorar productos, filtrarlos y agregarlos al carrito.

## 🧠 Descripción

Este proyecto consiste en una interfaz de tienda online donde los usuarios pueden:

- Visualizar un listado de productos
- Filtrar y/o buscar productos
- Ver detalles individuales
- Agregar productos al carrito
- Gestionar cantidades dentro del carrito

## ⚙️ Instalación y ejecución

1. Clonar el repositorio:

```
git clone https://github.com/FranQuarnolo/shopListing.git
```

2. Entrar en el proyecto:

```
cd shopListing
```

3. Instalar dependencias:

```
npm install
```

4. Ejecutar el proyecto:

```
npm run dev
```
